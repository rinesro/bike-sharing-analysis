# Bike Sharing Analysis Dashboard 🚲

## Deskripsi
Proyek ini merupakan analisis data dari dataset Bike Sharing untuk memenuhi tugas akhir Dicoding.

## Struktur Folder
- `/dashboard`: Berisi `dashboard.py` dan dataset.
- `/data`: Berisi dataset `day.csv` dan `hour.csv`.
- `notebook.ipynb`: Berkas analisis data lengkap.

1. Pastikan Python sudah terinstal.
